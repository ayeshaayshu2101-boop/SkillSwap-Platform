# Skill Swap & Micro-Learning Platform

## Project Overview

A web platform where users can exchange skills, book learning sessions, and provide ratings and reviews.

## Features

* User Registration
* User Login
* Dashboard
* Add Skills
* Search Skills
* Profile Page
* Session Booking
* Ratings & Reviews

## Technologies Used

* Python Flask
* SQLite
* HTML
* CSS

## Setup Instructions

1. Clone the repository
2. Install Flask

pip install flask flask-sqlalchemy

3. Run the application

python app.py

4. Open browser

http://127.0.0.1:5000

## Database

SQLite Database: skillswap.db
